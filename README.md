# ProtobufSerializer


### ProtobufSerializer

Implement C extension to pack dictionary to protobuf and write to file.

### Requirements

 - Python 3.5+
 - Docker image of CentOS 8

### Using

Your only need to download start.sh.

Start docker:

```
docker run -ti --rm -v <directory>:/tmp/otus/ centos /bin/bash
```  

Run:

```
sh start.sh
```
